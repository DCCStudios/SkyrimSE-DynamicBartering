This folder should contain a web font for the PrismaUI interface.

Recommended fonts (use one that matches Skyrim's aesthetic):
- Futura Condensed (Skyrim's actual UI font - proprietary, not distributable)
- Rajdhani (free Google Font, similar condensed geometric style)
- Josefin Sans Condensed (free, close to Futura's feel)

To use: download a .woff2 or .ttf and update barter_offer.css @font-face rule.

The CSS currently falls back to system fonts: 'Futura Condensed', 'Arial Narrow', 'Segoe UI'
